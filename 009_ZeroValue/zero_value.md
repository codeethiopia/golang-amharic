#### የዜሮ(የባዶ) ዋጋ
* የዜሮ ወይም የባዶ ዋጋ ምንድነው ? understanding zero value 
    * false for booleans (ቡሊያን ፣ ሐሰት)
    * 0 for integers (ኢንቲጀር ፣ 0)
    * 0.0 for floats (ፍሎት ፣ 0.0)
    * "" for strings (ስትሪንግ ፣ "")
    * nil (ኒል) for
        * pointers
        * functions
        * interfaces
        * slices
        * channels
        * maps

*use short declaration operator as much as possible*
